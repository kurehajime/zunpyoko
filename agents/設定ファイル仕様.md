# 設定ファイル仕様（zunpyoko）

対象ファイル: `public/*/config.json`

## ルート
- `vvprojPath` (string): VOICEVOXプロジェクトファイル名（`public`配下、例: `kaerunouta/voicebox.vvproj`）
- `wavPath` (string): 再生する音声ファイル名（`public`配下、例: `kaerunouta/kaerunouta.wav`）
- `activeMergeGapFrames` (number): アクティブ区間を連結する許容フレーム数
- `characters` (array): キャラクター設定の配列

## characters の要素
- `trackName` (string, optional): 参照するトラック名（未指定なら先頭トラック）
- `noteRange.min` / `noteRange.max` (number): 対象ノート番号の範囲
- `imagePath` (string): 通常時の画像（`public`配下のファイル名）
- `activeImagePath` (string, optional): アクティブ時の画像（`public`配下のファイル名）
- `position.x` / `position.y` (number): 画像の左下基準の位置（px）
- `width` / `height` (number): 表示サイズ（px）

## メモ
- `wavPath` / `imagePath` / `activeImagePath` は `public` 配下に置く。
- `activeMergeGapFrames` はノート間の隙間がこのフレーム数以下なら連続扱いにする。
